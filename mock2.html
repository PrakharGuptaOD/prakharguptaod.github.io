Mock Practice Questions 2 – Full Answer Sheet

1. PricePredictor — Preventing Overfitting in Real Estate Price Prediction using Regularization Techniques

Q1. What is the main objective of Regularization in ML Models?
Answer: b) Reduce Overfitting
Explanation: Regularization methods add penalties to the loss function in order to shrink and balance coefficient values, preventing the model from memorizing noise in the training data.
Q2. Which Regularization Technique is used for Feature Selection?
Answer: b) Lasso
Explanation: Lasso Regression (L1 regularization) tends to force some coefficients exactly to zero, thereby performing implicit feature selection.
Q3. In Ridge Regression, what happens to the feature coefficients?
Answer: b) They are shrunk towards zero but never become zero
Explanation: Ridge Regression (L2 regularization) penalizes large coefficients by shrinking them but does not eliminate any feature completely.
Q4. Lasso Regression minimizes:
Answer: c) Sum of Squared Errors + L1 Penalty
Explanation: Lasso minimizes the combination of the sum of squared errors and an L1 penalty term which encourages sparsity in the coefficients.
Q5. Which metric is often used to tune Regularization strength (λ)?
Answer: c) Cross-Validation Score
Explanation: Cross-validation is typically used to determine the optimal λ (or alpha) that balances bias and variance to achieve the best generalization.
MSQ Q6. Overfitting in ML occurs when:
Answer: a) Model learns noise from data; c) Model performs poorly on unseen data
Explanation: Overfitting happens when a model fits both the underlying pattern and the noise within training data, resulting in poor generalization to new data.
MSQ Q7. Regularization helps in:
Answer: a) Reducing variance; c) Preventing overfitting
Explanation: Regularization techniques reduce model variance, which in turn helps prevent overfitting, at the cost of a small increase in bias.
MSQ Q8. Elastic Net is preferred over Lasso when:
Answer: b) Features are highly correlated
Explanation: Elastic Net combines L1 and L2 penalties and works better than Lasso when there is high multicollinearity among features.
MSQ Q9. Higher values of λ in Regularization will:
Answer: b) Shrink coefficients more; c) Lead to underfitting if too large
Explanation: Increasing λ makes the regularization penalty stronger, which shrinks coefficients further. If λ is too high, the model may become too simple (underfitting).
MSQ Q10. Select Regularization Techniques used in Machine Learning:
Answer: a) L1 Regularization; b) L2 Regularization
Explanation: L1 (Lasso) and L2 (Ridge) regularizations are the most common types used to prevent overfitting in models.

Q4. Which method replaces NaN values in a DataFrame column?

Q5. Which operation is vectorized in both NumPy and Pandas?

MSQ Q6. What are benefits of using Pandas over plain Python lists for data analysis?
Answer: a) Faster performance on large datasets; b) Built-in functions for analysis; d) Easy CSV and Excel handling

3. HeartRisk Predictor — Health Risk Classification using Logistic Regression

Q1. What activation function is used in Logistic Regression?
Answer: c) Sigmoid
Explanation: Logistic Regression uses the sigmoid function to squash outputs between 0 and 1.
Q2. What is the output range of the Sigmoid function?
Answer: b) 0 to 1
Explanation: The sigmoid function outputs probabilities in the range of 0 to 1.
Q3. Logistic Regression is most suitable for:
Answer: c) Classification Problems
Explanation: Logistic Regression is used to classify data into discrete categories.
Q4. If predicted probability = 0.8, and threshold = 0.5, the predicted class is:
Answer: b) 1
Explanation: When the predicted probability exceeds the threshold of 0.5, the model classifies the observation as 1 (High Risk).
Q5. Which of the following is NOT an evaluation metric for binary classification?
Answer: c) R² Score
Explanation: R² Score is used for regression problems, not for binary classification.
MSQ Q6. Advantages of Logistic Regression include:
Answer: a) Simple and interpretable; b) Works well with linearly separable data; c) Can model multi-class problems (with extensions)
Explanation: Logistic Regression is valued for its simplicity and interpretability, and with extensions (like One-vs-Rest), it can also handle multi-class scenarios.
MSQ Q7. Components of a Confusion Matrix for binary classification include:
Answer: a) True Positive (TP); b) True Negative (TN); d) False Positive (FP) (and implicitly False Negative (FN))
Explanation: A confusion matrix summarizes the counts for TP, TN, FP, and FN.
MSQ Q8. Which techniques help prevent overfitting in Logistic Regression?
Answer: a) L1 Regularization; b) L2 Regularization; d) Cross-validation
Explanation: Regularization (both L1 and L2) and cross-validation help maintain a model that generalizes well to unseen data.
MSQ Q9. Which of the following affect the performance of Logistic Regression?
Answer: a) Irrelevant features; b) Feature scaling; c) High correlation among features
Explanation: The inclusion of irrelevant features, poor scaling, or collinearity among features can negatively affect model performance.
MSQ Q10. In which cases might Logistic Regression fail?
Answer: a) Highly non-linear data; c) When classes are imbalanced; d) When data has a high level of noise
Explanation: Logistic Regression models a linear decision boundary; non-linear patterns, imbalanced classes, or noisy data can limit its effectiveness.

4. BookFinder — Efficient Search in a Digital Library

Q1. Which condition must be true for Binary Search to work correctly?
Answer: b) Data must be sorted
Explanation: Binary Search requires that the dataset be sorted in order to divide the search interval.
Q2. What is the worst-case time complexity of Linear Search?
Answer: c) O(n)
Explanation: Linear Search may check each element, yielding a worst-case time complexity of O(n).
Q3. What is the best-case time complexity of Binary Search?
Answer: c) O(1)
Explanation: If the target element is at the middle position on the first try, Binary Search achieves O(1) time.
Q4. What is returned if the key is not found in Binary Search?
Answer: b) -1 or "Not Found"
Explanation: When an element is not found, Binary Search typically returns -1 or a similar indicator.
Q5. Linear Search is more suitable when:
Answer: c) Data is unsorted
Explanation: Linear Search does not require data to be sorted, making it suitable for unsorted collections.
Q6. How many comparisons (approx.) will Binary Search need for 1024 elements in worst case?
Answer: a) 10
Explanation: Binary Search divides the dataset roughly in half with each comparison; log2(1024) = 10 comparisons.
Q7. Binary Search follows which strategy?
Answer: b) Divide and Conquer
Explanation: Binary Search divides the dataset into halves repeatedly, a classic divide and conquer approach.
Q8. Which of the following is not true about Linear Search?
Answer: c) Works only with numbers
Explanation: Linear Search can be applied to any list regardless of data type; the other statements are true.
Q9. What is the mid-point index in Binary Search for start = 0, end = 9?
Answer: b) 4
Explanation: The mid index is floor((0+9)/2) = 4.
Q10. Why might Binary Search perform poorly if data is unsorted?
Answer: d) Both it may enter an infinite loop and give wrong output
Explanation: Binary Search assumes sorted data; unsorted data can lead to incorrect midpoints and faulty logic.

5. ModelMetrics — Evaluating ML Models for Health & Finance Sectors

Q1. Which metric should be used when false positives are more dangerous than false negatives?
Answer: c) Precision
Explanation: Precision measures the proportion of positive identifications that were actually correct; thus, when false positives are more harmful, precision is key.
Q2. If a classification model has high recall but low precision, it means:
Answer: b) It makes many false positives
Explanation: High recall indicates most actual positives are identified, but low precision means many of those identified are false positives.
Q3. The metric that gives equal importance to precision and recall is:
Answer: c) F1-Score
Explanation: The F1-Score is the harmonic mean of precision and recall, giving equal weight to both.
Q4. Which metric is most suitable for a regression task?
Answer: c) RMSE
Explanation: RMSE (Root Mean Squared Error) quantifies the error magnitude for regression models.
Q5. If R² score = 1, what does it indicate?
Answer: c) Perfect prediction
Explanation: An R² score of 1 means the model perfectly explains the variability in the response data.
Q6. What does a confusion matrix show?
Answer: c) Count of TP, TN, FP, FN
Explanation: A confusion matrix tabulates the true positives, true negatives, false positives, and false negatives.
Q7. Which metric is calculated as: TP / (TP + FN)?
Answer: b) Recall
Explanation: Recall quantifies the proportion of actual positives that are correctly identified by the model.
Q8. Which error metric penalizes larger errors more than smaller ones?
Answer: b) MSE
Explanation: MSE (Mean Squared Error) squares the error term, thus penalizing larger deviations more than smaller ones.
Q9. Which of the following is not a classification metric?
Answer: d) RMSE
Explanation: RMSE is used for regression tasks, not for classification.
Q10. What is the range of R² score?
Answer: d) -∞ to ∞
Explanation: In practice, R² can be negative when the model fits the data very poorly and up to 1 for a perfect model.

Q2. Polynomial Regression is useful when:
Answer: c) Relationship is non-linear
Explanation: Polynomial Regression fits data where the relationship between the independent and dependent variables is curved.
Q3. In Polynomial Regression, what is added to the model?

Explanation: Polynomial Regression augments the model by including higher-degree terms of the input features.
Q4. Which of the following can result in overfitting in Polynomial Regression?
Answer: c) Very high degree of polynomial

Q5. Which metric helps compare both models to see which fits the data better?

Explanation: The R² Score measures the proportion of variance explained by the model; higher values indicate a better fit.
Q6. Linear Regression assumes the target variable changes:
Answer: b) Linearly with input

Q7. What transformation is needed before applying Polynomial Regression?

Q9. If the R² score is 0.92 for Linear and 0.97 for Polynomial Regression, what can be said?

7. EvalSuite — Evaluating ML Models for Multiple Domains

Q1. Which metric is best to evaluate a clustering model like K-Means?
Answer: b) Silhouette Score
Explanation: The Silhouette Score measures how similar an object is to its own cluster compared to other clusters.
Q2. What does MAE (Mean Absolute Error) measure?
Answer: c) Average of absolute prediction errors
Explanation: MAE calculates the average absolute difference between the predicted and actual values.
Q3. Which metric is calculated as: (2 × Precision × Recall) / (Precision + Recall)?
Answer: b) F1-Score
Explanation: This formula is the harmonic mean of precision and recall, known as the F1-Score.
Q4. What is Reconstruction Error mostly used for?
Answer: c) Measuring error in unsupervised autoencoders
Explanation: Reconstruction Error quantifies the difference between the original input and its reconstruction, commonly in autoencoder models for anomaly detection.
Q5. When false positives must be minimized (e.g. spam filtering), which metric is most important?
Answer: b) Precision
Explanation: Precision focuses on the accuracy of positive predictions, which is crucial when false positives are costly.
MSQ Q6. Which of the following are classification metrics?
Answer: a) Accuracy; b) Recall; d) F1-Score
Explanation: Accuracy, recall, and F1-score are all metrics used in evaluating classification models.
MSQ Q7. Select all that are regression evaluation metrics:
Answer: a) MAE; b) MSE; d) R² Score
Explanation: MAE, MSE, and R² Score are widely used metrics for regression tasks.
MSQ Q8. When is Recall more important than Precision?
Answer: a) Cancer detection; d) Fraud transaction detection
Explanation: In scenarios like cancer detection or fraud detection, missing a positive case (low recall) is more critical than having some false alarms.
MSQ Q9. What are properties of Silhouette Score?
Answer: a) Ranges between -1 and 1; b) Measures how well points fit into clusters
Explanation: A Silhouette Score near 1 indicates that data points are well-clustered, while values near -1 indicate poor clustering.
MSQ Q10. Reconstruction error can be used to:
Answer: a) Detect anomalies; c) Determine outliers in autoencoders
Explanation: Reconstruction error is useful for identifying anomalies when the reconstruction of normal data is much better than that of outliers.
